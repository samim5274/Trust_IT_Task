# 🌐 Trust-IT (BD) Ltd – Landing Page (Technical Assessment)

## 📌 Project Overview

This project is a **responsive company landing page** developed as part of the **Junior Frontend Developer Technical Assessment** for **Trust-IT (BD) Ltd**.

The goal of this task was to demonstrate practical frontend development skills including clean UI design, responsive layout, structured code, and basic interactivity using modern web technologies.

---

## 🎯 Objective

To design and develop a visually appealing and fully responsive landing page for a fictional IT company, ensuring:

* Clean and modern UI
* Proper semantic HTML structure
* Responsive design across devices
* Interactive elements using JavaScript
* Organized and maintainable code

---

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript

### Optional Enhancements

* Bootstrap / Tailwind CSS *(if used)*
* CSS animations & transitions

## 📄 Features & Sections

### 🔹 1. Hero Section

* Company logo/name
* Eye-catching heading
* Short description
* Call-to-action button

### 🔹 2. Services Section

* Displays multiple service cards
* Each card includes:

  * Icon/Image
  * Service Title
  * Brief Description

### 🔹 3. About Section

* Company introduction
* Supporting image/illustration

### 🔹 4. Contact Form

* Fields:

  * Name
  * Email
  * Message
* JavaScript form validation:

  * Required field validation
  * Email format checking
  * Error message display

### 🔹 5. Responsive Navigation Bar

* Fully responsive design
* Hamburger menu for mobile devices
